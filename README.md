# Generative AI Assignment

## Overview
This project demonstrates text generation using a pretrained GPT model. The goal was to explore how generative AI works and how different parameters impact text output.

## Tools Used
- Python
- Google Colab
- Hugging Face Transformers

## Features
- Text generation using prompts
- Control of output length
- Temperature tuning for creativity
- Top_k and Top_p sampling

## How to Run
Open the notebook in Google Colab and run all cells step by step.
